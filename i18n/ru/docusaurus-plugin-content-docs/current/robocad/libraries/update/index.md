# Обновление образа и библиотек

Это руководство показывает, как обновить образ и библиотеки **robocadV** на реальных роботах.  

:::warning
Перед обновлением библиотек убедитесь, что образ операционной системы поддерживает новые версии библиотек. Лучше обновить образ вместо ручного обновления библиотек!
:::
