# Описание этапов запуска проекта  

- Скачайте приложение [shufflecad](../shufflecad) из [SoftHub](../softhub);  
- [Установите необходимые компоненты для выбранного языка программирования](download-required);  
- [Создайте начальный проект и сконфигурируйте его](create-project);  
- Дополните программу [работой с датчиками и моторами](../robocad/libraries/algaritm);  
- Запустите *shufflecad* и [заполните необходимые поля для запуска](../shufflecad/start-project).
