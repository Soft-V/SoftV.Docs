# Добро пожаловать в shufflecad

**shufflecad** — это рабочий стол для робота.
С его помощью можно запускать пользовательские программы, наблюдать за их жизнью и управлять ими так же легко, будто двигаешь детали в конструкторе.
Приложение открывает визуальную панель управления — здесь переменные обретают форму индикаторов и регуляторов, которые можно менять прямо на ходу и использовать для отладки.

**shufflecad** показывает взгляд робота на мир — поток с камеры и результаты обработки изображений, позволяя увидеть то, что видит и «понимает» робот. Все действия, ошибки и решения робота собираются в удобном логере, превращая неясные сообщения в понятную историю выполнения программы.

Это не просто инструмент: **shufflecad — интерфейс между идеями разработчика и поведением робота, позволяющий видеть, регулировать и запускать интеллект машины.**
