# Логи 

Для просмотра событий, происходящих в приложении, и для просмотра логов из программы используется страница **Логи**.  
<div style={{textAlign: 'left'}}>
<img src="/docshome/img/shufflecad/shufflecad_3.png"/>
</div>   

Сверху представлены элементы для удобной работы со страницей:  
- Количество записей;  
- Переключатель для прокрутки к последнему сообщению при его добавлении;  
- Кнопка для сохранения всех сообщений в файл;  
- Кнопка для очистки логов.

</div>
