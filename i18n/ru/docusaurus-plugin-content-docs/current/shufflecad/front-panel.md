# Мониторинг  

Для мониторинга состояния робота используется страница **Мониторинг**. Здесь Вы можете управлять переменными состояния, которые Вы добавили у себя в программе.  
<div style={{textAlign: 'left'}}>
<img src="/docshome/img/shufflecad/shufflecad_2.png"/>
</div>  

Справа отображается список переменных, которые можно перетягивать на основное поле для работы с ними:   
<div style={{textAlign: 'left'}}>
<img src="/docshome/img/shufflecad/shufflecad_2_drag.png"/> 
</div>   

Снизу есть кнопки для работы с панелью элементов:  
- Кнопка очистки всех элементов на панели;  
- Кнопка для импорта элементов панели;
- Кнопка для экспорта элементов панели.    
<div style={{textAlign: 'left'}}>
<img src="/docshome/img/shufflecad/shufflecad_2_low_buttons.png"/> 
</div>   

Для сортировки и фильтрации переменных в списке есть специальные элементы управления:   
<div style={{textAlign: 'left'}}>
<img src="/docshome/img/shufflecad/shufflecad_2_sort.png"/> 
</div>
