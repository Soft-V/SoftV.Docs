# Геймпад  

Для работы с геймпадом/джойстиком используется страница "Геймпад". Здесь вы можете управлять виртуальным геймпадов, если реальный не подключен к комплютеру:  
<div style={{textAlign: 'left'}}>
<img src="/docshome/img/shufflecad/shufflecad_5.png"/>
</div>   

При подключении настоящего джойстика/геймпада к компьютеру - виртуальный геймпад будет работать только как индикатор, управлять им будет нельзя.
