# 欢迎使用 shufflecad

**shufflecad** 是面向机器人的桌面环境。
使用它可以运行用户程序、观察程序的运行情况并管理程序，就像在构建器中移动零件一样简单。
该应用提供了可视化控制面板 —— 变量变为指示器和控制器，可在运行时直接调整，用于调试。

**shufflecad** 展示了机器人看世界的视角 —— 摄像头流和图像处理结果，让您看到机器人“看见”和“理解”的内容。所有操作、错误和决策都会记录到便捷的日志器中，将难以理解的信息转化为清晰的执行历史。

这不仅仅是一个工具：**shufflecad 是开发者的想法与机器人行为之间的接口，让您能看到、调整并运行机器的智能。**
