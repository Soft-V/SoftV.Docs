# 事件日志

“事件日志”页面用于查看应用程序中发生的事件以及程序日志。
<div style={{textAlign: 'left'}}>
<img src="/docshome/img/shufflecad/shufflecad_3.png"/>
</div>

顶部提供了便于操作页面的控件：
- 记录条数
- 在新消息到达时自动滚动到最新消息的切换开关
- 将所有消息保存到文件的按钮
- 清空日志的按钮
