# 项目启动阶段说明  

- 为 **VMX** 和 **Titan QUAD** 控制器下载固件（说明在 robocadV 模拟器的课程中提供）；
- [下载镜像并将其安装到 SD 卡](../robocad/libraries/update/image)；
- 从 [SoftHub](../softhub) 下载 [shufflecad](../shufflecad) 应用程序；  
- [为您选择的编程语言安装必需的组件](download-required)；  
- [创建初始项目并进行配置](create-project)；  
- 补充程序 [处理传感器和电动机](../robocad/libraries/studica)；  
- 运行 *shufflecad* 并 [填充必要的字段以启动](../shufflecad/start-project)。
