# 启动项目的步骤

- 从 [SoftHub](../softhub) 下载 [shufflecad](../shufflecad) 应用；  
- [为所选编程语言安装必要的组件](download-required)；  
- [创建并配置初始项目](create-project)；  
- 添加用于传感器和电机的代码（[Algaritm 库](../robocad/libraries/algaritm)）；  
- 运行 *shufflecad* 并[填写启动所需的字段](../shufflecad/start-project)。
