# Description of Project Launch Stages  

- Download the firmware for the **VMX** and **Titan QUAD** controllers (instructions are provided in the lessons in the robocadV simulator);
- [Download and install the image to the SD card](../robocad/libraries/update/image);
- Download the [shufflecad](../shufflecad) application from [SoftHub](../softhub);  
- [Install the necessary components for your chosen programming language](download-required);  
- [Create an initial project and configure it](create-project);  
- Supplement the program with [work with sensors and motors](../robocad/libraries/studica);  
- Run *shufflecad* and [fill in the necessary fields to start](../shufflecad/start-project).
